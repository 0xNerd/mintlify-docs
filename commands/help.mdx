## Overview

The `/help` command provides a condensed guide to using ZK Send, including all available commands and links to documentation.

## What you'll see

When you run `/help`, the bot sends:

- **Quick command reference**: A list of all available commands with brief descriptions
- **Documentation links**: Direct links to the getting started guide and FAQ
- **Support contact**: Button to contact operations team at `ops@zksend.app`

## When to use

Use `/help` when you:

- Forget which commands are available
- Need a quick refresher on how to use the bot
- Encounter an issue and want to find relevant documentation
- Want to contact support

## Command quick reference

The help message includes:

| Command | Purpose |
| ------- | ------- |
| `/start` | Initialize or restore your account |
| `/balance` | Check your wallet balance |
| `/send` | Send SOL privately |
| `/wallets` | Manage your wallets |
| `/import` | Import an existing wallet |
| `/delete` | Remove your account |
| `/help` | Show this help message |

## Getting more help

If `/help` doesn't answer your question:

- Check the full documentation linked in the help message
- Review the detailed command pages for each command
- Contact support via the button in the help message or email `ops@zksend.app`

<Note>
  The bot also uses `/help` as a fallback when it can't parse free-form user questions.
</Note>
